
# 🧮 Calculín Vue

**Calculín** is a simple and effective price comparison calculator built with [Vue.js](https://vuejs.org/). Originally created to compare the price per meter of toilet paper rolls 🧻, the tool has evolved into a general-purpose calculator that helps you compare *anything* — rolls, bottles, packages — and find the best value for your money.

## 🚀 Live Demo

Try it out here:  
👉 **[https://elegant-dango-8ee614.netlify.app/](https://elegant-dango-8ee614.netlify.app/)**

## 💡 Features

- Compare two products by **price** and **unit quantity**
- Get instant feedback on which option offers the better price per unit
- Minimal and responsive UI
- Built with **Vue 3** and **TailwindCSS**

## 🧑‍💻 Development Setup

Clone the repository:

```bash
git clone https://github.com/achavazza/calculinVue.git
cd calculinVue
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
